# 📊 포트폴리오 분석 리포팅 워크플로우

> 목표  
> 사용자가 보유 종목을 입력하면 → 현재 상태 평가 → 매도/추가매수/보유 전략 → 애널리스트 의견 → 종합 투자 결론까지 자동 생성하는 리서치 기반 포트폴리오 분석 시스템 구축

---

## 1. 입력 데이터 수집 (User & Market Data Layer)

- 사용자 입력 데이터
  
  - 종목 코드 / 종목명
  - 매수가 / 매수 수량 / 매수일
  - 추가 매수 이력
  - 투자 성향 (단기/중기/장기, 공격/중립/보수)
  - 목표 수익률 / 허용 손실률

- 시장 데이터 연동
  
  - 현재가, 전일 대비, 거래량
  - 과거 가격 데이터 (일/주/월)
  - 업종 지수, 시장 지수 (KOSPI/KOSDAQ)
  - 재무제표 (분기/연간)
  - 밸류에이션 지표 (PER, PBR, ROE 등)
  - 애널리스트 리포트, 목표주가 컨센서스
  - 뉴스, 공시, 이벤트 일정

---

## 2. 포트폴리오 요약 지표 생성 (Dashboard Summary Engine)

- 총 투자액 계산
- 현재 평가액 계산
- 총 수익금 / 수익률(%) 계산
- 포트폴리오 구성 종목 수 집계
- 성과 등급 산출 (매우 우수 / 우수 / 보통 / 주의)

---

## 3. 개별 종목 카드 생성 (Stock Card Generator)

- 종목별 카드 자동 생성 항목
  - 현재가
  - 매수가
  - 평가금액
  - 보유 수량
  - 수익금 / 수익률
  - 성과 상태 배지 (강세 / 중립 / 약세)
  - “상세 분석 보기” 버튼 연결

---

## 4. 종목 상세 분석 엔진 (Detail Analysis Engine)

### 4-1. 요약 탭 (Summary Engine)

![서비스 대시보드 + 요약](./서비스%20대시보드_1.png)

- 현재 성과 평가 문장 생성
- 목표가 대비 현재 위치 평가
- 종목 특성 요약
  - 사업 구조
  - 산업 포지션
  - 배당 성향
- 밸류에이션 상태 요약
- 포트폴리오 내 비중 표시

---

### 4-2. 매도 타이밍 탭 (Exit Timing Engine)

![매도타이밍](./서비스%20대시보드_2_매도타이밍.png)

- 1차 익절 전략
  
  - 목표가
  - 매도 수량
  - 예상 수익
  - 기술적/펀더멘털 근거
  - 권장 시점

- 2차 익절 전략
  
  - 최종 목표가
  - 잔여 물량 매도 전략
  - 리스크 요인 명시
  - 예상 보유 기간

- 단계적 익절 추천 문구 자동 생성

---

### 4-3. 추가 매수 탭 (Accumulation Engine)

![추가매수](./서비스%20대시보드_3_추가매수.png)

- 추가 매수 가능 구간 산출
  - 지지선 기반 가격대
  - 밸류에이션 저평가 구간
- 권장 추가 매수 비중(%)
- 매수 시점 조건
  - 실적 발표 전/후
  - 기술적 반등 신호 시
  - 시장 조정 구간

---

### 4-4. 익절/손절 탭 (Risk Control Engine)

![익절/손절](./서비스%20대시보드_4_익절손절.png)

- 손절선 자동 계산
  - 가격 기준 손절선
  - 포트폴리오 손실 한도 기준 손절선
- 주요 리스크 요인 요약
- 방어 전략 제시
  - 부분 매도
  - 비중 축소
  - 헤지 전략

---

### 4-5. 매매 전략 탭 (Trading Strategy Engine)

![매매전략](./서비스%20대시보드_5_매매전략.png)

- 현재 시장 국면 판단
  - 추세 추종
  - 박스권
  - 변동성 확대 구간
- 권장 전략 유형
  - 단기 스윙
  - 중기 보유
  - 장기 투자
- 포트폴리오 내 역할 정의
  - 핵심 성장주
  - 방어주
  - 레버리지 포지션

---

### 4-6. 보유 기간 탭 (Holding Horizon Engine)

![보유기간](./서비스%20대시보드_6_보유기간.png)

- 권장 보유 기간 제시
  - 단기 / 중기 / 장기
- 근거 제시
  - 산업 사이클 단계
  - 실적 사이클 위치
  - 거시 환경 (금리, 경기, 정책)

---

### 4-7. 애널리스트 탭 (Analyst Insight Engine)

![애널리스트](./서비스%20대시보드_7_애널리스트.png)

- 주요 증권사 리포트 요약
  - 투자 의견 (Strong Buy, Buy 등)
  - 목표주가 범위 및 평균
  - 상·하방 여력 분석
- 핵심 논거 요약
  - 실적 전망
  - 산업 성장 요인
  - 기술 경쟁력
- 연간/분기별 전망 타임라인 자동 생성

---

## 5. 종합 평가 & 투자 결론 생성 (Final Verdict Engine)

- 투자 등급 산출 (1~5점)
- 종합 평가 문장 자동 생성
- 현재 추천 전략 정리
  - 부분 익절 + 장기 보유
  - 수익 일부 확보 후 리스크 관리
  - 산업 사이클 상승 수혜 유지

---

## 6. 포트폴리오 리밸런싱 엔진 (Rebalancing Engine)

- 종목 비중 과다/과소 자동 감지
- 업종 편중 여부 분석
- 레버리지/고위험 자산 비중 경고
- 추천 리밸런싱 시나리오 생성
  - 특정 종목 축소 → 다른 종목 확대

---

## 7. 리포트 자동 생성 & 저장 (Auto Reporting System)

- 리포트 형식
  
  - 웹 대시보드
  - PDF 다운로드
  - 주간/월간 리포트 자동 발송

- 리포트 구성
  
  1. 포트폴리오 요약
  2. 종목별 상세 분석
  3. 매도/추가매수/보유 전략
  4. 애널리스트 의견 요약
  5. 종합 투자 결론
  6. 향후 모니터링 체크리스트

---

## 8. 실시간 알림 & 모니터링 시스템

- 목표가 도달 알림
- 손절선 근접 경고
- 실적 발표/중대 뉴스 알림
- 포트폴리오 리스크 급증 시 경고
