# PWA (Progressive Web App) 통합 및 아이콘 설정

## 개요
사용자가 웹사이트를 데스크탑 및 모바일 기기에 앱처럼 설치하여 사용할 수 있도록 PWA 기능을 통합하고, 전용 아이콘을 생성/적용함.

## 구현 상세

### 1. 앱 아이콘 생성
- **원본 소스**: `public/App_icon_3.png`
- **처리 방식**: `sharp` 라이브러리를 사용하여 원본 로고를 트리밍하고, **흰색 둥근 사각형 배경** 위에 합성하여 세련된 앱 아이콘 형태로 변환.
- **최종 파일**:
  - `public/icon-192x192.png`: Android/PC 설치용 (소형)
  - `public/icon-512x512.png`: Android/PC 설치용 (대형)
  - `public/apple-icon.png`: iOS 홈 화면용 (180x180)
  - `public/favicon.png`: 브라우저 탭 및 파비콘용 (32x32)

### 2. Manifest 설정
- **파일**: `public/manifest.json`
- **설정 내용**:
  - 앱 이름: StockPilot
  - 아이콘 설정: 표준 파일명 적용 및 `purpose: "any maskable"` 지원.
  - 시작 URL: `/` (클린 URL 적용)

### 3. Next.js PWA 설정
- **라이브러리**: `@ducanh2912/next-pwa`
- **설정 파일**: `next.config.ts` (PWA 플러그인 연동 완료)

### 4. Layout 메타데이터
- **파일**: `src/app/layout.tsx`
- **내용**: 최종 생성된 아이콘 파일들을 메타데이터에 등록하여 브라우저가 인식하도록 설정.

## 확인 방법
1. 개발 서버 재시작 (`npm run dev`)
2. Chrome 브라우저 접속 -> 주소창 우측 '설치' 아이콘 클릭
3. 설치된 앱 실행 시 투명 배경 아이콘 확인
